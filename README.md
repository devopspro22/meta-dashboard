# דשבורד קמפיינים פייסבוק

דשבורד מקצועי לניטור ביצועי קמפיינים ב-Meta Ads (Facebook / Instagram).

מציג: הוצאות · המרות · ROAS · CTR · CPM · CPL · גרף יומי  
מתעדכן אוטומטית כל 15 דקות.

---

## 📊 מדדים מוצגים

| מדד | תיאור |
|---|---|
| **הוצאות** | סך הוצאות בתקופה הנבחרת |
| **המרות** | מספר לידים / שיחות WhatsApp |
| **CPL** | עלות להמרה |
| **ROAS** | החזר על ההשקעה |
| **CTR** | אחוז הקלקות |
| **CPM** | עלות ל-1000 חשיפות |
| **Reach** | טווח הגעה |

---

## 🛠️ הרצה מקומית

```bash
npm install
npm start
```
